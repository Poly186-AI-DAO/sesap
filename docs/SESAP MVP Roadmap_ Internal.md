_Written by Princeps_

# Overview

The SESAP MVP Roadmap outlines the incremental development plan for the self-executing social agreements platform. Starting with the simple UI for agreement creation, execution and monitoring, then gradually adding more advanced features like customizable contract templates, document management, and organization management, this roadmap covers all the key functionalities that you want to offer to the users. The final phase includes features like translation, recommendations AI, and discovery engine to make SESAP a complete platform for executing social agreements.  
<br/>The MVP will allow customers to create, execute, and monitor agreements through a simple user interface, with added features and improvements along the way. By the end of the MVP phase, customers can expect a platform that allows for e-signatures, the ability to upload their driver's license for trustworthiness, the minting of contracts and agreements on the blockchain, customizable contract templates, and improved organization management through cloud talent and document management solutions.

SESAP MVP aims to provide customers with a simple UI for agreement creation, execution, and monitoring in Phase 1, with additional features such as e-signature functionality and document verification through the VERTEX VISION API.  
<br/>In Phase 2, the platform will mint contracts and agreements on the blockchain, offer customizable contract templates, and analyze uploaded contracts using DOCUMENT AI. Phase 3 will focus on creating company and organization structures, defining job postings, and improving the UI.  

Phase 4 will bring cloud talent-powered features to handle organization management in SESAP, using both GPT-3 and Codex APIs to test applicants' understanding of organizations. Phase 5 will introduce the DocAI Warehouse for document management, intelligent products, and a discovery engine to increase conversion rates. Finally, in Phase 6, the platform will offer translation and language detection for agreements, custom recommendations, and a cross-industry discovery engine.

  
Please note that the roadmap is subject to change.

# Roadmap

## Phase 1:

### Simple Agreement Creation

- Simple UI for agreement creation, execution, and monitoring  
    
- E-Signature functionality  
    
- Upload drivers license using the VERTEX VISION API  
    

## Phase 2:

### Contract Minting and Analysis

- Minting Contracts and Agreements on the Blockchain  
    
- Customizable Contract Templates  
    
- Analyzing uploaded contracts and agreements with DOCUMENT AI  
    
- Improved UI for agreement creation, execution, and monitoring  
    

## Phase 3:

### Organization Management

- Creation of Companies and Organization Structures  
    
- Define and generate job postings for vacant positions  
    
- Improved UI for agreement creation, execution, and monitoring  
    

## Phase 4:

### Cloud Talent Powered Features

- Cloud Talent Powered Features for organization management in SESAP  
    
- Use of GPT-3 and Codex APIs to test applicants' understanding  
    
- Improved UI for agreement creation, execution, and monitoring  
    

## Phase 5:

### DocAI Warehouse for Document Management

- Provision of DocAI Warehouse for document management and organization  
    
- Intelligent products for personalized ownership experiences  
    
- Enable users to experience organization, project, or product discovery  
    
- Improved UI for agreement creation, execution, and monitoring  
    

## Phase 6:

### Improved User Experience

- Translation with language detection and translation for all agreements and contracts  
    
- Recommendations AI for contracts, agreements, organizations, projects, and jobs  
    
- Discovery Engine for cross-industry discovery solutions  
    
- Improved UI for agreement creation, execution, and monitoring
